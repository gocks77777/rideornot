## OpenClaw 작업 기록 (2026-02-18)

이 문서는 OpenClaw 인공지능 어시스턴트(자비스)와 협업하여 진행한 작업들을 기록합니다.

### 1. Raspberry Pi 환경 설정 및 패시브 인컴 노드 배포 시도

*   **Docker 설치:** Raspberry Pi (ARM64)에 Docker를 성공적으로 설치했습니다.
*   **Honeygain 배포 (성공):** Honeygain Docker 컨테이너를 성공적으로 배포하고 설정하여 수익 창출을 시작했습니다. (대시보드 확인 완료)
*   **Pawns 배포 (성공):** Pawns Docker 컨테이너를 성공적으로 배포하고 설정하여 수익 창출을 시작했습니다. (로그 확인 결과 수익 발생 중)
*   **Nillion 배포 (실패/제거):** ARM64 아키텍처 비호환성 문제로 Docker 컨테이너 실행에 실패하여 제거했습니다.
*   **Gaganode 배포 (실패):** Docker 컨테이너 및 Meson Network 공식 설치 스크립트를 통한 직접 설치 모두 실패했습니다. ARM64 호환성 또는 설치 과정 문제로 추정됩니다.
*   **기타 노드 배포 (실패):** Titan Edge, Blockmesh, Gradient Sentry, Grass 노드 또한 ARM64 아키텍처 비호환성 또는 이미지 접근 문제로 배포에 실패했습니다.

### 2. 가상 선물 거래 봇 개발 및 백테스팅 (King-Bot)

*   **King-Bot 스크립트 개발:** 바이낸스 선물 시장에서 Supertrend 및 ADX 지표를 활용한 가상 거래 봇 (`paper_king.py`)을 개발했습니다. MDD 및 매매 횟수 기록 기능을 추가했습니다.
*   **전략 작동 확인:** 현재 King-Bot은 백그라운드에서 실행 중이며, 로그를 통해 숏 포지션 진입 및 포지션 유지, 수익 발생 등 정상적인 거래 활동을 확인했습니다. (현재 숏 포지션에서 미확정 수익을 기록 중)
*   **3년 장기 백테스팅 시도 (실패):** 2023년부터 2026년까지의 3년치 1분봉 데이터로 백테스팅을 시도했으나, Binance API의 대량 데이터 요청 제한 문제로 인해 데이터 가져오기에 실패했습니다. 이로 인해 백테스팅이 완료되지 못했습니다.

### 3. 웹 앱 프로젝트 분석 (`rideornot`)

*   **프로젝트 클론:** `https://github.com/gocks77777/rideornot` GitHub 저장소를 OpenClaw 워크스페이스에 성공적으로 클론했습니다.
*   **프로젝트 개요:** Next.js 기반의 웹 앱으로 보이며, Supabase를 백엔드로 사용하고 지도/위치 기반 기능(예: 차량/택시 공유 서비스)을 포함하는 것으로 파악됩니다.
